# Proyecto Primer Parcial - Desarrollo de Aplicaciones Web (DAW 6-7) 🚗💨

## 📌 Tema: Sitio Web Informativo e Interactivo - "El Transportador"
Este proyecto consiste en el diseño y desarrollo de una aplicación web del lado del cliente dedicada a la película de acción *El Transportador* (protagonizada por Jason Statham).

---

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructuración de contenido mediante etiquetas (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **CSS3:** Estilos personalizados centralizados en un archivo externo, aplicando Flexbox, CSS Grid Layout, transiciones interactivas y diseño responsivo (*Media Queries*).
* **JavaScript:** Programación del comportamiento dinámico en el entorno del cliente.

---

## 🚀 Características del Sitio Web

El sitio está compuesto por **una página principal y dos páginas adicionales**, distribuidas de la siguiente manera:

1. **Inicio (`index.html`):** * Encabezado temático y banner de presentación (*Hero section*).
   * Datos importantes y sinopsis de la obra cinematográfica.
   * **Componente Interactivo (JavaScript):** Galería/Carrusel dinámico (*Slideshow*) con controles manuales y transición automática de imágenes de acción de la película.
2. **Personajes y Reglas (`personajes.html`):**
   * Detalle dinámico de las 3 reglas inquebrantables de Frank Martin.
   * Fichas técnicas del elenco principal estructuradas estéticamente mediante un diseño de rejilla flexible (*CSS Grid*).
3. **Contacto (`contacto.html`):**
   * **Formulario con Validación en el Cliente:** Formulario de solicitud de servicios con validación en tiempo real programada en JavaScript (control de campos vacíos, expresiones regulares para correos electrónicos seguros y manejo dinámico de alertas de error).

---

## 📱 Responsividad y Diseño Adaptativo
El diseño ha sido optimizado con un enfoque *responsive*, garantizando una navegación fluida, legibilidad de textos y escalado correcto de componentes visuales tanto en pantallas de escritorio como en dispositivos móviles.
